# BIOS

- It stands for - Basic Input Output System

- It runs first when the **Power Button** is pressed

  > power ON -----> BIOS starts -----> POST ------> load Bootloader -----> kernel starts
  
  - **POST** --> (Power On Self Test) It checks hardware like ram, cpu, keyboard, disk, gpu etc
 
  - BIOS boots the first 512 bytes of the data in the disk
